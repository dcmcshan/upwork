🎯 QUICK COMMANDS
─────────────────

Setup:
  ./install.sh                    Install dependencies & setup
  python get_access_token.py      Get Upwork API access token
  ./upwork test                   Test API connection

Monitor:
  ./upwork search                 Search with default queries
  ./upwork search --query "X"     Search for specific skill/keyword
  ./upwork search --continuous    Run continuous monitoring

Filter:
  ./upwork filter --skill python  Filter by skill
  ./upwork filter --min-rate 100  Filter by pay rate

Analyze:
  ./upwork analyze                Full insights report
  ./upwork analyze --skill X      Skill-specific analysis
  ./upwork analyze --export X.json Export insights

Stats:
  ./upwork stats                  Show cache statistics

🎓 USE CASES
────────────

1. Job Hunting
   Find best opportunities matching your skills
   → ./upwork search --query "your skill"
   → ./upwork filter --min-rate 75 --list

2. Market Research
   Track skill demand and pay rates over time
   → ./upwork search --continuous --interval 120
   → ./upwork analyze --export weekly.json

3. Rate Benchmarking
   Determine competitive rates for your services
   → ./upwork search --query "your specialty"
   → ./upwork analyze --skill "your specialty"

4. Career Planning
   Identify high-value skills to learn
   → ./upwork search
   → ./upwork analyze
   → Check "HIGHEST PAYING SKILLS" section

5. Competition Analysis
   Monitor specific technology markets
   → Edit .env: SEARCH_QUERIES=AI,blockchain,web3
   → ./upwork search --continuous

🔧 CONFIGURATION
────────────────

File: .env

Required:
  UPWORK_CLIENT_ID          Your Upwork OAuth2 client ID
  UPWORK_CLIENT_SECRET      Your Upwork OAuth2 secret
  UPWORK_ACCESS_TOKEN       Your OAuth2 access token

Optional (Job Monitor):
  SEARCH_QUERIES            Comma-separated search terms
  MIN_HOURLY_RATE          Minimum hourly rate to track
  MAX_HOURLY_RATE          Maximum hourly rate to track
  MIN_FIXED_PRICE          Minimum fixed price to track
  MAX_FIXED_PRICE          Maximum fixed price to track

Optional (Auto-Apply Agent):
  JOB_KEYWORDS             Keywords for auto-application
  MIN_BUDGET               Minimum budget for auto-apply
  MAX_BUDGET               Maximum budget for auto-apply
